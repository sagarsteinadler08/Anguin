# Anguin - Advanced SBMI Calculator

**Anguin** is a next-generation Body Mass Index calculator that goes beyond the standard height/weight ratio. It works by integrating:
- **Age**: Recognizing that older adults benefit from higher weight reserves.
- **Ethnicity**: Adjusting risk thresholds for Asian, South Asian, Black, and other populations based on global health data.
- **Gender**: Accounting for body composition differences.

## Features
- **Smart BMI (SBMI)**: Scientifically adjusted interpretations.
- **3D Biomechanics**: Live 3D visualization of human movement.
- **Interactive UI**: "Gravity" theme with neon aesthetics and glassmorphism.
- **Privacy First**: All calculations run locally in your browser.

## How to Run
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   streamlit run app.py
   ```

## Tech Stack
- Python 3.9+
- Streamlit
- Pandas
- Three.js (for 3D visualization)
